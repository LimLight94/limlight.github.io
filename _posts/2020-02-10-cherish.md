# Cherish


### clean architecture

1. Dagger2(depncency injection) 적용

2. MVVM

3. RxJava2

4. Retrofit2

### description

1. recyclerView 이용 소식페이지

2. 스케쥴러 이용 가구의 행동 스케쥴링 

3. 가구 모드의 커스텀 기능

4. iot 가구의 동작 제어(EspBlufiForAndroid 사용)
